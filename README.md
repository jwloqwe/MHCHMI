# MHCHMI

## Instructions

1. Start cloning the repo down to your local machine
2. Run "npm i"
3. You should be able to run "npx cypress open"
4. Once you have run the cypress command, you should be able to see the Cypress UI
5. Click on Start for Chrome or any Preferred browser
6. You should be able to see the test case that is written in ~/cypress/e2e/BasicFeatureSpec.cy.js
7. Click on the file to run the test case
8. You should be able to see the executed test cases.
